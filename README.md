# Olympics-Data-Analysis-
Analysis of historical Olympic medal data using Excel, Oracle SQL, and Power BI to identify trends in country performance, athlete achievements, gender participation, and sport-wise dominance.

## Tools & Technologies
- Microsoft Excel (Data Cleaning)
- Oracle SQL (Data Storage & Analysis)
- Power BI (Data Visualization)

- ## Key Analysis Performed
- Analyzed country-wise medal distribution to identify top-performing nations
- Examined medal trends over the years to understand performance patterns
- Identified top 10 athletes based on total medal count
- Studied gender-wise medal distribution to assess participation trends
- Evaluated sport-wise medal contribution to determine dominant sports

## Visualizations
Interactive dashboards were created in Power BI using bar charts, line charts, and pie charts to clearly present insights and trends.

## Outcome
The project demonstrates practical application of data preprocessing, SQL-based querying, and business intelligence visualization for real-world data analytics.
